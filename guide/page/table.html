컴포넌트 가이드

테이블 (Table)

스타일

  • 텍스트 데이터와 명목 데이터는 왼쪽 정렬

    일반 텍스트 데이터와 범주를 나타내는 명목 데이터(예: 날짜, 우편번호, 핸드폰 번호, IP 주소 등)

  • 양적 데이터는 오른쪽 정렬

    숫자 데이터(예: 개수, 퍼센트, 용량 등)는 오른쪽으로 정렬

  • 빈 데이터 셀에는 대시(-) 기호 사용

    데이터가 없는 경우, 셀을 비워두지 않고 대시(-) 기호를 사용

항목 텍스트 정렬(align)
1 기본 (default), 강조 (.em), 중요 (strong) -
2 텍스트 데이터와 명목 데이터 align-a
3 양적 데이터 align-b
3 - align-b
