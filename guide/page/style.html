색상(Color)

색상은 비즈니스 브랜딩의 핵심 구성 요소이며, 사용자가 사이트와 상호작용할 때 중요한 메시지를 전달하는 데 도움이 됩니다.

Primary color 주요 색상은 브랜드의 아이덴티티를 나타내며, UI 전체의 주요 구성 요소에 사용됩니다.

  • primary-005 #536dfe
    (83, 109, 254)
  • primary-010 #3d5afe
    (61, 90, 254)
  • primary-020 #2b46db
    (43, 70, 219)

Secondary color 보조 색상은 앱의 상호작용 요소에 사용되는 색상으로, 활성화 상태를 표시하는 데 사용됩니다.

  • secondary-005 #ff9e90
    (255, 158, 144)
  • secondary-010 #ff6e40
    (255, 110, 64)
  • secondary-020 #ff3d00
    (255, 61, 0)

Base color
(Grayscale) 기본 색상은 정보의 계층을 표현하는 데 사용됩니다. 배경(BG), 텍스트(Text), 선(Line) 등에 적용되며, 특히 텍스트는 가독성을 위해 배경과의 명암비를 고려해야 합니다.

  • base-010 #121212
    (18, 18, 18)
  • base-020 #363636
    (54, 54, 54)
  • base-030 #686868
    (104, 104, 104)

System color 사용자에게 시스템 상태를 직관적으로 알리는 데 사용됩니다. 예) 오류(error), 경고(warning), 성공(success) 상태를 나타내는 색상이 있습니다.

  • system-010 #d32f2f
    (211, 47, 47)
  • system-020 #f9a825
    (249, 168, 37)
  • system-030 #4caf50
    (76, 175, 80)
  • system-040 #0091ea
    (0, 145, 234)
